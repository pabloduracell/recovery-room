# Paciente WebGL Experience

Mini experiencia web interactiva con React + Three.js + React Three Fiber.

## Ejecutar en local

```bash
npm install
npm run dev
```

Abre la URL que te dé Vite.

## Publicar

Sube el proyecto a GitHub y conéctalo a Vercel o Netlify.

Build command:
```bash
npm run build
```

Output directory:
```bash
dist
```
